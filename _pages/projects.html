Projects

RoutineMe

Technologies: Android, Java

RoutineMe is a native Android application written in Java designed to log workout routines, add timed exercises to routines, and run them using a built-in timer. It utilizes Model-View-ViewModel (MVVM) architecture for effective design, with Room Persistence Library and SQLite to store and retrieve application data. It's great for workouts where you need to constantly set and reset timers (ex: three sets of planks, each one minute, with a one minute rest in between), as this application automates it for you. RoutineMe also has automated tests, end-to-end testing and continuous integration for quality assurance.

amphibNames

Technologies: Node.js, Pug.js

amphibNames is a full stack web application written in Node.js and Pug.js designed to update hundreds of historical amphibian taxonomies at once, with results temporarily saved to disk with options for the user to download. amphibNames works by querying two online databases via GET requests and parses HTML and text file responses to extract new taxonomies. With the taxonomies of amphibians constantly being updated through new research, this web application streamlines the process of updating taxa pulled from earlier literature.

Flights R' Us

Technologies: Node.js, ExpressJS, Vue.js, Sequelize.js, PostgreSQL

Flights R’ Us is a flight booking web application developed by myself and three other teammates. We designed, refined and normalized E/R diagrams and database schemas for conceptual modeling, and executed our plans by creating and implementing multiple raw SQL queries for customers and staff users. Customers can login to a booking system to search for flights by different criteria, view flight and seat information, and book seats. Additionally, customers can view, update and cancel their own reservations. Through a staff account, flight information and bookings are able to be updated. Information on flight and seat data are stored in a database, and are queried through user input.

insightUBC

Technologies: TypeScript, Node.js

insightUBC is a web application written by myself and another teammate through a scrum-based, agile process over the course of 5 sprints. In this application, we developed a backend user language to query results in JSON so that users are able to view and query metadata from all of the courses offered at the University of British Columbia. This includes room locations, statistics on grades/grade distributions, and common aggregate operations (min, max, sum, etc.) similar to SQL queries. I also implemented REST endpoints for the frontend to work with, and ensured quality with unit and integration tests and by applying TDD.

Buses R' Us

Tools/APIs: Android, Java

Buses R’ Us is a mobile Android application designed to help users plan their trips via public transit accordingly. Real time data from the TransLink API is queried and parsed to display bus stops, bus routes, and bus arrival/departure times across the lower mainland in BC. In this project, I developed the backend models, business logic, and frontend drawables.
