Projects

Flights R' Us

Tools/APIs: Javascript, PostgreSQL, Sequelize, ExpressJS, Vue.js, Nuxt.js

Flights R’ Us is a flight booking application where customers can login to a booking system to search for flights by different criteria, view flight and seat information, and book seats. Additionally, customers can view, update and cancel their own reservations. Through and administrative account, flight information and bookings are able to be updated. Information on flight and seat data are stored in a database, and are queried through user input.

insightUBC

Tools/APIs: Typescript, Javascript

insightUBC is a web application where users are able to view and query metadata from all of the courses offered at UBC, including room locations, statistics on grades/grade distributions, and common aggregate operations (min, max, sum, etc.) similar to SQL. This involved backend development in parsing course data in JSON and room/location data in HTML, and implementing REST endpoints frontend.

Buses R' Us

Tools/APIs: Java, Genymotion, VirtualBox

Buses R’ Us is a mobile Android application designed to help users plan their trips via public transit accordingly. Real time data from the TransLink API is queried and parsed to display bus stops, bus routes, and bus arrival/departure times across the lower mainland.
